JavaScript Day 2

Invocation function

Invocation function is calling a function using objects or directly writting the function like this There are six different ways to invoke a function and they are

  1. Direct Invocation:calling the function by it's name
  2. Method Invocation:calling the function through object
  3. Constructor Invocation:calling the function through object but the object and function name will be same
  4. Indirect Invocation:calling the function indirectly using call, apply, and bind
  5. Immediately Invoked Function Expression:calling the function Immediately
  6. Arrow Function Invocation:declaring functions as Expressions using arrows

Direct Invocation

calling the function using thr function name

The code will be like this,
function directinvocation(a,b)
{
let c=a+b;
document.getElementById("di").innerHTML= c;
}
directinvocation(20,20);

Method Invocation is calling function using object to create object for a function we need to decalre the function inside a variable i.e, object,like this
let addtion={
methodinvoction:function(a,b){
let c=a+b;
document.getElementById("mi").innerHTML=c;

}
addtion.methodinvoction(30,30);
Here addtion is the object inside that object methodinvoction is declared like this addtion.methodinvoction(30,30); we invoke a methodinvoction function using dot operator

Constructor Invocation

Constructor Invocation where the function will be called using objects the code will be like this,
function Constructorinnvocation (a,b){
this.c=a+b;
}
let d= new Constructorinnvocation(40,40);
document.getElementById("ci").innerHTML=d.c;
Here the new keyword creates a object i.e d and this keyword creates a object inside a function i.e c using these objects we invoke a function

Indirect Invocation

indirect Invocation is calling the function indirectly using call,apply and bind keyword usage of this keyword will differs based on the variable which we are going to use for object variables we use call the example code,
function indirectinvocationc(a,b)
{
let sum=a+b;
document.getElementById("iic").innerHTML=sum;
} let c={a:40,b:50};
indirectinvocationc.call(c,c.a,c.b);

Here the c is a object variable so we use call keyword to call the function indirectinvocationc

for array variables we use apply the example code,
function indirectinvocationa(a,b,d)
{
let sum=a+b+d;
document.getElementById("iia").innerHTML=sum;
}
let j=[40,60,10];
indirectinvocationa.apply(null,j);

Here the j is an array variable so we use apply keyword to call the function indirectinvocationa

Bind function is used to declare a function inside a variable
function indirectinvocationb(a,b)
{
sum=a+b;
document.getElementById("iib").innerHTML=sum;
}
let iib=indirectinvocationb.bind(null,60,30);
iib();

Here the iib variable has been binded with the function indirectinvocationb using bind keyword

Arrow function

Arrow function is declaring functions as Expressions like this

const add = (a, b) => a + b;
document.getElementById("af").innerHTML=add(8,8);

Including this there are some more functions and they are

  1. Anonymous Function:These functions are passed as a arguments or declared in a variable
  2. Callback Function:These functions are passed as a arguments to another and then they are called in that function
  3. Generator function:The Generator function is a recursive function
  4. Async function:Async function waits and check for the requirements and then runs the function declared in the await
