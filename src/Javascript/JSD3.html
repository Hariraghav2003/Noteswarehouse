JavaScript Day 3

Objects!!

An object is like a reak time which has several properties,Ex:Car it's a object which has several properties such as break accel clutch headlight color etc..
const car={type:"maruti",model:"suzuki",color:"grey"}
this is the sample code where car is the object which has 3 properties such as type,model,color and each has values maruti,suzuki,grey.

Javascript object properties

The Javascript object properties can be accessed or modified in below ways,

  1. Accessing properties of object: The properties of the object can be accessed in 3 ways but the most common is using dot(.) operator
    const car={type:"maruti",model:"suzuki",color:"grey"}
    console.log(car.color);
    The car.color access the car object's color property using dot operator and prints the value using console.log
  2. Adding new properties: to add new properties we code like this
    car.cc="1220cc"
    where cc is a new property of car which is added to car like car.cc="1220"
  3. Deleting properties: to delete new properties we code like this
    delete car.cc;
    here the newly added cc property is deleted using delete keyword

Javascript object methods

The Javascript object methods are declaring function as an object property like this
car.carname= function(){
return this.type+""+this.model;
};
console.log(car.carname());
here car name is an object method because it's declared as a function.Like this we can declare by adding or we can declare when we define the object. Important note: we must use this keyword while handling objects inside a function.

Javascript object display

The Javascript objects can be displayed in 4 ways

  1. Object values:It display the values of object
  2. Object enteries:It display the object with their property and values
  3. Stringyyfy:displays object in JSON format
  4. loop:used to diplay the object which has more properties using for loop
