JavaScript Day 4

JavaScript Events!!

JavaScript events are used to handle the behaviour of a HTML page using JavaScript for example if we click or if we perform any action it should perform an event there are many event handlers in an HTML but the most used JavaScript events are,

  • Onclick():If we click a html element it should perform a function
  • ondblclick():If we double click a HTML event it should perform a function
  • onmouseover():If move the mouse over a HTML event it should perform a function
  • onchange()

Onclick


If we click the above button it will trigger mynamecl function

< button > onclick="mynamecl()">Click the button < /button >
function mynamecl(){
document.getElementById("hello").innerHTML="Hello I am Hariraghav";
}

In the above we have declared the button element with a onclick event so if we click that button it will trigger the function declared inside the onclick event keyword

Ondblclick


If we double click the above button it will trigger mynamedcl function

< button > onclick="mynamedcl()">Double Click the button < /button >
function mynamedcl(){
document.getElementById("hi").innerHTML="Hello I am Hariraghav";
}

In the above we have declared the button element with a ondblclick event so if we double click that button it will trigger the function declared inside the ondblclick event keyword

Onmouseover


If we Hover through the mouse it will trigger mynameover function

< button > onclick="mynameover()"> Hover through the mouse < /button >
function mynameover(){
document.getElementById("over").innerHTML="Hello I am Hariraghav";
}

In the above we have declared the button element with a onmouseover event so if we hover that button it will trigger the function declared inside the onmouseover event keyword
