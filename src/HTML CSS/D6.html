CSS Day 1

Linking CSS in a HTML file

3 ways to link CSS:Inline,Internal,External

Inline CSS is styling HTML Page in the tag itself this content is styled by inline CSS Method

Internal CSS is styling HTML page in the html file using style tag in the header tag part of html file and this achieved by using 3 methods such as Selector & Universal Selector(*): The slector is achived by using Tag name h1tag,Ptag,etc.. the universal selector applies defined styles on all the tags present in the html file.

Class: In class the styles are applyed using the dot (.) operator.This Content is styled by class method.

ID:In ID the styles are applyed using the "#" operator. This content is styled by ID method.

External CSS is styling HTML page by using a CSS file and linking it in the HTML web page using link tag(link rel="stylesheet" href="file path of the CSS") this method is same as Internal CSS but the differnce is the styles are defined in seperate file and stored in .CSS extension this content is styled using External CSS method

Using these methoda we can style a HTML pages in different ways
