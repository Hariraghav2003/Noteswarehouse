CSS combinators

CSS combinators are used to select and style the tags there are 4 ways to select the tags based on the symbols and they are

  • space-descendent selector:selects all the elements of the specified tag
  • >-child selector:selects the child of the specified tag which means if div>p means the div is the parent and p is child so it styles all the div tag childs(p)
  • +-adjacent sibling selector:selects the immediate child of the parent which means if div+p means the div elements immediate tag (p) will be styled
  • ~-general sibling selector:it's same as adjacent selector but the difference is it will color all the immediate element of the child
