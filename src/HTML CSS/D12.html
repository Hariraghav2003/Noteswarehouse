CSS Shadows

CSS shadows are used to create a shadow effect for the contents there are two types of shadow effect

Text shadow

Text shadow is used to create shadow effect for the text

Ex text-shadow:2px 2px 2px red

Box Shadow

Box shadow is used to create shadow effect for images or text as box like this

box-shadow: 10px 10px;

additionally we can import different styles of box shadow from CSS Scan website through this we can create like this

CSS Texteffects

CSS text effect is styling the text content there are 4 properties in text effects and they are

Text-overflow

Text overflow is controling the overflowing text which has two methods:clip,elipsis where clip hiddes the overflow content whereas elipsis make the overflow content hidden and repalce it with dots

Word wrap

word wrap shorts the overflow content and make it fit in the box based on the methods(break word,break all,keep all)

writing mode

In writing mode we can change the text writing order like left to right ,right to left

CSS 2D transform

2D transform where we can chnage the X axis and Y axis using these methods

  • transalate:moves the element from its current postion in X and Y axis based on the parameters
  • rotate:where we can rotate the content in X and Y axis
  • scaleX:method increases or decreases the width of the element in X axis
  • scaleY:method increases or decreases the width of the element in Y axis
  • Scale:shorthand property for ScaleX and ScaleY
  • skewX squeeze the element in X axis
  • skewY squeeze the element in Y axis
  • skew:shorthand property for skewX and skewY
