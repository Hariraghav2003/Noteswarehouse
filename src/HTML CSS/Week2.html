In day 7 we learnt how to style borders and backgrounds

Styling the borders has four main properties such as

  • Border-style: used to create varities of border for the contents
  • Border-color:used to color the borders created using border-style
  • Border-width:used to increase the width of the borders
  • Border-radius:used to smoothen or sharpen the border edges

Also a shorthand property border: using this we can do the above listed properties in a single line

Styling the background has two properties

Background-color
Background color is used to set the color for the background for a whole page or for a tag alone
Background-image
To set the background of the image
background image as three propertie
Background-postion: to set the postion for the backgroundimage
Background-repeat:used to repeat the background image either in X axis or in Y axis.it's default if the background image size is small
Background-attach:used to fix the background image if we scroll the image
