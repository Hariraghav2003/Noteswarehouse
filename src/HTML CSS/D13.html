CSS Transition

CSS Transition is used for changing the properties smoothly like this

Transition

CSS transition has 4 properties

1.Transition-delay

The transtion delay specifies the delay for the transtion effect for example

Transition delat count is 5

2.transition-duration

transition-duration is used to set the time for the transtion effect

transition-duration is 8s

3.Transition property

Transition property is used to set the property where the transtion effect should effect if width property is set it will effect in width property

In this box the width only has transtion effect because transtion property is set as width

4.Transition timing functions

Transtion tming function are used to set speed of transition effect there are of 5 types

  • ease:default value the transtion effect will be slow start and fast and ends slowly
  • linear:transition speed will be same from start to end
  • ease-in:transition effect will be slow start
  • ease-out:transition effect will be slow end
  • ease-in-out:transition effect will be slow start & slow end

and cubic-bezier to define our own speed

CSS Animation

css animation is used to move or create a animation effect

In CSS animations there is a important property known as key frame where we define the animation

CSS Animation has 8 properties

  • animation-name: animation name is the name for the animation for the content which we are going to animate
  • animation-duration: to set the duration for the animation
  • animation-delay:specifies a delay for the animation
  • animation-iteration count:to set how many times the animations wants to continue
  • animation-direction:to set the directions for the animation (normal,reverse,alternate,alternate reverse)
    • normal:
    it's a normal flow of animation forwards
    reverse:
    it's a reverse flow of animation backwards
    alternate:
    The flow of animation will be forwards then backwards
    alternate-reverse:
    The flow of animation will be backwards then forwards
  • animation-timing-function:the animation function is same as transition-timing-function but this is used for animations
