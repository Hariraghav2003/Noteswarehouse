CSS Links

The links created using anchor tag can be styled in many ways.This anchor links has 4 states and they are,

  • Link:it's a default link an unvisited link created using anchor tag can be styleed using this a:link
  • visited:if user visited a link then that particular visited links can be styled using a:visited this syntax
  • hover:if user moves the cursor over the link it shows a differnt style using a:hover this syntax
  • active:if user clicked the cursor then that state can be styled using the syntax a:active

Example styling for each states of link

this link is styled on 4 link states
  • if the link is unvisited it will be green
  • if the link is visited it will be red
  • if we hover over this link it will be blueviolet
  • if the link is active it will be chartreuse

Like this we can style the background color of the 4states of links using Background-color: inside the needed state

  2. list-style-postion:Used to specify the postion of the bulletin in a list
  3. List-style-image:to provide image as the bulletin for the list

  • Table Alignment
  • Table Style
  • Table Responsive

Table Style

The above seen are basic styles or needed one to form a table but this table style are used to enhance the apperance of the Table.This table Style has 4 properties

  • Table Padding
  • Horizontal Dividers
  • hoverable table
  • striped table

Stripped tables

Stripped Tables are used to style each rows of the table using nth-child(even),nth-child(odd)

Table Responsive

Responsive table are used to make the table responsive by providing a horizontal scroll bar if the screen size is small using the property overflow-x:auto
