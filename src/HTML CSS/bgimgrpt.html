Background Image Repeat

The Background image is repeated by default in horizontal or vertical direction if the background image size is small,if we want to repeat the picture size in vertical or horizontal we can use background image repeat property ,

Background image repeat-x
To repeat it horizontally
Background image repeat-y
To repeat it vertically
We can fix the postion of the background image in the right,left,top,bottom,center using background postion property the top right jpeg image is an example for using background-position property
