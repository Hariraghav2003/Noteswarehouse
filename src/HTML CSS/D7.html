CSS Borders

CSS borders has 4 basic properties:

  • Border Style
  • Border Width
  • Border Color
  • Special Borders(Rounded Borders)

Border Style

Border Styling is the basic of border css has more designs of borders such as dotted,solid and etc.. The basic syntax is Border-style:needed style and we can also set a mixed styled border by providing 4 values like this border-style: dotted solid dashed groove; where

  1. dotted is top line
  2. solid is right line
  3. dashed is bottom line
  4. groove is left line
like this it applies styles in a manner of top,right,bottom,left if we provide 3 values like removing Solid style it will apply style in a manner of
  1. dotted as top line
  2. dashed as right & left line
  3. groove as bottom line
like this it applies styles in a manner of top,right,left,bottom if we provide 2 values like removing solid and dashed style it will apply style in the manner of
  1. dotted as top & bottom line
  2. groove as right & left line
like this it applies styles in a manner of dotted(top & bottom) groove (right&left) this is the universal pattern of border styles for all the border properties

border-style: dotted solid dashed groove

Border Width

Border width defines the width of the borders The basic syntax is border-width:needed width

Ex:border-width: 5px

Border Color

Border width defines the width of the borders The basic syntax is border-color:needed Border-color

Ex:border-color:red
