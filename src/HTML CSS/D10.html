CSS Postion

The contents in the html page are fixed in a particular place using postion property there are 5 postion property

  • Static: To fix the text in a particular postion and it will not adjust based on the contents
  • Sticky:it's same as static but the difference is it will adjust based on the content
  • Absolute:it's like a wall where we can right a content and it will move based on the web page
  • Relative:It's like a poster in wall where we can fix the postion of the contents inside the absolute
  • FIxed:it's same as ststic but it will scroll or move if the content moves

CSS Z index

Z index is used to set the layers where we can set the postive and negative values if we set 1 it will be 1st layer if we set 2 it will be second layer if we set -1 it will be set behind the 1st layer, for example

CSS Overflow

The CSS overflow is used to fit the over sized content inside a box there are 4 ways to fit the content and they are

  • view:it will display the overflow text
  • hidden:it will hide the overflow contents
  • auto:it will provide the automatic needed scrolling axis option
  • scroll:it will provide the scroll option in both axis

CSS display

CSS display property is of three types,

  • display:inline-it displays the inline elements in the same line but the difference is it will affect the horizontal space and the width,height will be same as the content
  • display:block-it displays the inline elements in the next line but the difference is it will affect the height and width of the contents
  • display:inline-block-it displays the inline elements in the sameline but the difference is it will affect the height and width of the content also we can customize the height and width
