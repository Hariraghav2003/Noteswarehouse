Bootstrap Day 2

In Bootstrap there is a core concept known as layouts where we do all styling activites such as grid,mediaqueries columns etc..

Break points

Break points are pre defined mediaqueries in Bootstraps which are of 6 varaties,

The most used breakpoint is lg and sm

Containers

The Containers are like div which are used to create nav bar contents of web page and more with breakpoints there are 3 containers,

  • Container:which is the basic container with max width based on breakpoints
  • Container-fluid:which is a container with max width of 100% at all breakpoints
  • container-{break-point}:customaizable container-fluid based on breakpoints
